# MoneyMind — Personal Finance Dashboard

สิ่งที่สร้าง: เว็บแอปพลิเคชันจัดการการเงินส่วนตัวที่ใช้ AI ช่วยวิเคราะห์และให้คำแนะนำ

---

## 🎯 วิธีใช้

### 1. **หน้า Dashboard (ภาพรวม)**
- แสดงยอดคงเหลือ + Sparkline รายจ่ายในช่วง 30 วัน
- KPI 4 ตัว (รายรับ, รายจ่าย, ใช้งบ%, อัตราการออม) เทียบกับเดือนก่อน
- กราฟ Trend 30D/90D/1Y — อ่านจากธุรกรรมจริง, Label ตรงวันที่ล่าสุด
- บัตร "AI วิเคราะห์" → เสนอหมวดที่ใช้เพิ่มขึ้นมากสุด หรืออัตราการออม
- Donut chart ประเภทรายจ่าย + 6 ธุรกรรมล่าสุด

### 2. **หน้า Transactions (ธุรกรรม)**
- ตารางธุรกรรมทั้งหมด พร้อม:
  - ค้นหาตามชื่อร้าน
  - กรองตามประเภท (รับเงิน/ใช้เงิน)
  - จัดเรียงตามวันที่ หรือจำนวน
  - เลือกหมวดเฉพาะ

### 3. **หน้า Upload Statement (นำเข้า)**
- ลากไฟล์ Statement PDF (K-Bank/กสิกร) มาวาง
- Parser แยกวันที่, ร้าน, จำนวน, ประเภท โดยอัตโนมัติ
- AI จัดหมวดตามคำสำคัญ:
  - ร้านเมนูเสบียง/Starbucks → อาหาร
  - ปั๊มน้ำมัน/พีทีที → ขนส่ง
  - Lazada/Shopee → ช้อปปิ้ง
  - ทรูมูฟ/ค่าไฟ → บ้าน
  - เป็นต้น
- ไพรวิวรายการทั้งหมด → แก้ไข/ลบ ก่อนยืนยัน
- กด Import → บันทึกและซิงค์ Dashboard/Insights ทันที

### 4. **หน้า AI Insights (วิเคราะห์)**
- แสดงคะแนนสุขภาพการเงิน (0-100) + Rating (ดีมาก/ดี/พอใช้/ต้องระวัง)
- **โหมด Derived** (หลังนำเข้า):
  - ระบบสร้าง Insight Cards จาก txs จริง
  - หมวดโตสุด, อัตราการออม, ร้านซ้ำเยอะ, สัดส่วนใหญ่สุด
- **โหมด AI** (กด "วิเคราะห์ด้วย AI"):
  - เรียก Claude Haiku เพื่อวิเคราะห์ลึก
  - แสดงผล 4-6 Insight Cards พร้อมเสนอแนะ

### 5. **Chat กับ Mind**
- คลิกปุ่ม "คุยกับ Mind" ทั่วแอป
- ถามเกี่ยวกับการใช้เงิน Mind ตอบส่วนตัว + ให้คำแนะนำ

### 6. **Notifications (การแจ้งเตือน)**
- กดระดิ่ง (🔔) มุมบนขวา
- ดูประวัติแจ้งเตือน (เช่น "นำเข้า N รายการสำเร็จ")

---

## ⚙️ Tweaks (เซ็ตอัพ)

กดปุ่มสกรูมุมขวา เพื่อปรับ:
- **สีเน้น**: Champagne / Mint / Lilac / Amber
- **ความหนาแน่น**: Regular / Compact
- **ภาษา**: ไทย / English
- **สกุลเงิน**: THB / USD
- **แสงพื้นหลัง**: เปิด / ปิด

---

## 🔄 การซิงค์ข้อมูล

เมื่อนำเข้า Statement:
- **Dashboard**: Sparkline, กราฟ Trend, KPI deltas อัปเดตจากธุรกรรมจริง
- **AI Insights**: สร้าง Insight Cards ใหม่จากข้อมูลจริง (ล้างผล AI เก่า)
- **Notifications**: เพิ่มข้อมูล "นำเข้า N รายการ"
- **Transactions**: รายการใหม่ปรากฏเรียงจากล่าสุด

## 📊 ตัวอย่างวิธีใช้ครั้งแรก

1. **เข้าล็อก** (Demo account: demo@moneymind.app)
2. **ไปหน้า "นำเข้า Statement"** → ลาก Statement PDF มาวาง
3. **ตรวจสอบการจัดหมวด** ก่อนยืนยัน
4. **ดู Dashboard** → กราฟและคะแนนอัปเดตทันที
5. **ไป "AI Insights"** → เห็นการ์ดแนะนำจากข้อมูลจริง
6. **(ทางเลือก)** กด "วิเคราะห์ด้วย AI" เพื่อให้ Claude ลงลึก

---

## 🛠️ เทคนิก

- **Frontend**: React 18 + HTML/CSS (Dark Luxe aesthetic)
- **PDF Parser**: pdf.js (K-Bank statement support)
- **Category Auto-tagging**: Regex keyword matching (Thai + English)
- **Trend Calculation**: จาก txs จริง, anchor ที่วันธุรกรรมล่าสุด
- **AI Integration**: Claude Haiku 4.5 (opt-in analysis)
- **State Management**: React useState + useMemo (derived data re-sync on import)

---

## 📝 หมายเหตุ

- ข้อมูลหลักมาจากการนำเข้า Statement PDF เท่านั้น
- Dashboard/Insights ซิงค์อัตโนมัติเมื่อมีการเปลี่ยนข้อมูล
- AI analysis เป็นตัวเลือก — ระบบสามารถแสดง Insight ขั้นพื้นฐานจากข้อมูลจริงได้เสมอ
- หมวดจัดแบบ Heuristic (keyword matching) — ข้อความที่ไม่รู้จักจะ tag เป็น "other"
